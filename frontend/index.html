How to Use Median

1
Paste a URL

Copy any YouTube video or playlist, SoundCloud track or set, or Bandcamp album URL and paste it into the input field above.

2
Click Validate

Median will check the URL, detect the platform, and fetch the track or album info — title, artist, duration, and artwork.

3
Choose your format

Pick Audio Only (MP3 / FLAC / AAC), Video Only (MP4 / MKV / WebM), or Cover + Audio for SoundCloud/Bandcamp tracks with artwork embedded as video.

4
For playlists: merge or keep separate

Toggle Merge into single file to combine all tracks into one file with chapter markers — you can skip between songs inside the file.

5
Hit Download

The download runs in the background. Track it in the Active Downloads section. When done, the file is available for 15 minutes before auto-cleanup.

★
Watched Folder

Drop URLs (one per line) into watched/watched_urls.txt — Median will pick them up automatically every 5 seconds and start downloading.
